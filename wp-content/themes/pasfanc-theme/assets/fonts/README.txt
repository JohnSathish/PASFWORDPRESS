Oswald font - local hosting
===========================

To use Oswald-Regular locally (no external requests), place Oswald-Regular.woff2 in this folder.

Download options:
1. Google Fonts: https://fonts.google.com/specimen/Oswald -> Click "Download family" -> Extract and rename the Regular .ttf to use a converter for woff2, or use the .ttf if you update fonts.css
2. Or use an online woff2 converter on the .ttf from the Google Fonts zip
3. Or download pre-built woff2 from: https://cdn.jsdelivr.net/npm/oswald-webfont@0.0.1/fonts/Oswald-Regular.woff2
   Save the file as: Oswald-Regular.woff2 in this directory

If the file is missing, the theme falls back to loading from Google Fonts CDN.
